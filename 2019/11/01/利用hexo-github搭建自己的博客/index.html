hexo+github搭建自己的博客

2019-11-01

环境配置

下载 [git] (https://git-scm.com/downloads)  
下载 [Node.js] (https://nodejs.org/en/)
新建个文件夹作为博客文件夹，在文件夹中右键点击GitBash
安装 Hexo : $npm install -g hexo
安装依赖包： $npm install
博客文件夹初始化 hexo init

在github上建立仓库


注意，在Repository name中地址为 youname.github.io

github仓库赋予外网地址

打开自己的仓库，在Settings中找到GitHub Pages

None设置

将github与本机git连接

在git中


在连续回车后，打开C:\Users\Administrator.ssh地址

id_rsa.pub就是公钥，把里面东西复制下来，打开github

找到SSH and GPG keys
点击 New SSH key
在Key里输入你的公钥
点击添加
回到git
输入 ssh -T git@github.com

出现这个，即为连接成功
