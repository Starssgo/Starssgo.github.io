函数调用约定，是指当一个函数被调用时，函数的参数会被传递给被调用的函数和返回值会被返回给调用函数。函数的调用约定就是描述参数是怎么传递和由谁平衡堆栈的，当然还有返回值。

入栈规则(32位调用约定)

  • Cdecl

    参数入栈规则：从右到左
    还原栈：由调用者来修改栈

  • Stdcall

    参数入栈规则：从右到左。
    还原栈：由被调用函数来修改栈

  • Fastcall

    参数入栈：第一个跟第二个通过 ecx跟edx传递，剩余函数从右到左入栈
    还原栈：被调用者修改

{  
    B(x,y,z);  
}
